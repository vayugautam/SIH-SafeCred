### Color Palette
```
Primary Colors:
- Blue (#1e40af):        Primary actions, buttons
- Green (#10b981):       Approvals, success states
- Orange (#f59e0b):      Warnings, pending states
- Red (#ef4444):         Errors, fraud alerts, rejections

Background:
- Dark (#0f172a):        Main background
- Card (rgba(255,255,255,0.1)): Glassmorphism cards with backdrop blur

Text:
- White (#ffffff):       Primary text
- Gray-300 (#d1d5db):    Secondary text
- Gray-400 (#9ca3af):    Tertiary text
```

### Typography
```

- H1: 2.5rem (40px), font-weight: 700

- H3: 1.5rem (24px), font-weight: 600

- Small: 0.875rem (14px), font-weight: 400
- Tiny: 0.75rem (12px), font-weight: 400
```
